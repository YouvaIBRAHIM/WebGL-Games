Wing Impact

Vous prenez le contrôle du U-Wing avec lequel vous devez traverser un champ d'astéroïdes qu’il faudra éviter. 3 vies (max) vous sont attribuées, mais vous en perdrez une à chaque fois que vous toucherez un astéroïde. Le but est simple : obtenir le maximum de score possible. Que la Force soit avec vous !

How To Play

- Si le joueur appuie sur les touches Z/Q/S/D : contrôle du vaisseau U-Wing
- Si le joueur appuie sur la barre d’espace : le vaisseau accélère (ce qui permet de multiplier le score).
- Si le vaisseau touche un anneau : +2000 points (doublés à chaque nouveau niveau).
- Si le vaisseau touche un coeur : +1 vie.
- Si le vaisseau touche un astéroïde : -1 vie.
- Le vaisseau ne peut sortir du champ d'astéroïdes.
- Si vous finnissez le niveau, vous reviendrez au début et le vaisseau passera à une vitesse supérieure.
- Le score ne s'arrête pas entre les niveaux.
- Le score est est multiplié par x1.1 à chaque nouveau niveau.
- Les bonus récupérés ne seront plus visibles au niveau suivant (il faudra choisir le bon moment pour prendre un bonus)
